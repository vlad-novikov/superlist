Flutter to Google Sheet Demo App
Run Flutter Web App View Google Sheet View Code on GitHub
  • In this demo app, We have a Flutter app which takes Feedback input and it stores it in Google Sheet located in Google drive
  • To test it, Just open Flutter Web app from above option
  • Fill that fields and click button 'Submit Feedback'
  • After that Open Google Sheet from above option and you'll see your feedback is there appended in that Spreadsheet.
  • Also, you can see feedback responses in descending order after clicking button 'View Feedback'.
